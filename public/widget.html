Widget Options

Attribute Values Default Description
data-theme dark · light dark Widget color scheme — matches your site's background
data-compact true · false false Compact layout for sidebars; full layout for content areas
data-conflicts 3 · 5 · 8 · 10 5 Number of active conflicts to display in the widget
data-width any CSS value 100% Widget width — e.g. "300px" for a fixed sidebar
