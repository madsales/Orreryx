# OrreryX — Conversion Rate Optimization (CRO) Skills

> How to analyze and improve conversion rates across the OrreryX funnel.

---

## Conversion Funnel

```
Visitor → Landing Page → Trial Signup → First Login → Habit Formation → Paid Upgrade
```

Key metrics at each stage:
1. **Visit → Signup:** Target >5% (industry: 2–3%)
2. **Signup → First Login (24h):** Target >70%
3. **First Login → Day 7 Return:** Target >40%
4. **Trial → Paid:** Target >15%
5. **Paid → Month 2:** Target >70%

---

## Homepage CRO

### Hero Section
- Headline must pass the "5-second test": visitor knows what OrreryX does in 5 seconds
- Subhead adds credibility/context in 1 sentence
- CTA: "Start Free Trial" (no credit card friction)
- Hero image: risk map screenshot — shows, doesn't tell

### Social Proof Bar
Below the hero: logos, numbers, or short testimonials.
- "13 live conflict zones"
- "Trusted by investors in 40+ countries"
- Press logos if available

### Above-the-Fold Must Have
- [ ] Clear headline (what it is + benefit)
- [ ] Subhead with specificity
- [ ] CTA button with "no credit card" reassurance
- [ ] Visual showing the product
- [ ] Load time <2s

### Common Homepage Conversion Killers
- Vague headline ("The future of investing" = meaningless)
- No proof point above the fold
- Too many CTAs competing
- No "no credit card" copy near CTA
- Product screenshot missing or low quality

## Pricing Page CRO

- Always anchor Bloomberg/Stratfor price first
- Highlight recommended tier (Analyst) visually
- "No credit card" reassurance on every CTA
- FAQ section below pricing (handles objections without leaving page)
- Annual pricing toggle if available (converts better for high-intent visitors)

---

## Signup Flow CRO

Current: Email → magic link → login
- Minimize fields: email only (no name, no phone)
- Magic link reduces password friction
- Post-signup page: "Check your email" + expected delivery time
- If no open after 5 min: "Resend link" prompt

---

## Onboarding CRO (After First Login)

The first session is the highest-leverage conversion moment.

**Goal:** Get user to one "aha moment" as fast as possible.
- Aha moment: Seeing a live conflict event with market impact data
- Time to aha: target <90 seconds

**Onboarding flow:**
1. Land on dashboard → show most recent breaking event
2. Tooltip: "This event moved oil 1.8% today — see why →"
3. Show the market impact panel
4. Prompt: "Set up email alerts for events like this"

**Don't:**
- Show a blank state (no events)
- Ask for profile info before showing value
- Force a tutorial before showing the product

## Exit-Intent & Re-engagement

**Exit-intent popup (if user goes to leave pricing page):**
> "Wait — your free trial includes real-time data and email alerts. No credit card needed."
CTA: "Start for free"

**Abandoned signup (typed email but didn't submit):**
- If email captured: send magic link email with subject "your link is waiting"

## A/B Test Priority Queue

Run tests in this order (highest impact first):
1. Hero headline (biggest lever)
2. CTA button text ("Start Free Trial" vs. "Try Free — No Card")
3. Pricing page tier highlight
4. Email subject lines
5. Onboarding flow sequence
6. Social proof placement

See `ab-testing-skills.md` for test methodology.

---

## CRO Audit Checklist (Run Weekly)

- [ ] Check signup conversion rate (visits → signups)
- [ ] Check trial → paid rate
- [ ] Check Day 1 login rate (magic link open rate)
- [ ] Check Day 7 retention
- [ ] Review any drop-off point in Funnel
- [ ] Review last week's A/B test results
- [ ] Identify one hypothesis to test this week
